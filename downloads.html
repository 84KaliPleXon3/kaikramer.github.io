Downloads

Getting up and running with KeyStore Explorer is quick and easy. Everything you need to know is detailed below.

There are three KeyStore Explorer installation methods available:

  1. Windows Installation
  2. Mac OS X Installation
  3. Manual Installation

The first two are automated installs for Windows and Mac OS X. The third is a manual process suitable for all Java supported Operating Systems. Included below are full instructions for using each installation method.

Download Java Runtime Environment

Java Runtime Environment (JRE) Version 7 or above is required to run KeyStore Explorer. The latest JRE can be downloaded for free via the button below:

Windows Installation

  1. Download the installer for Windows with the button below.
  2. Double-click the kse-XX-setup.exe file to start the install.
  3. Follow the on-screen instructions to complete the install.
Download for Windows

Mac OS X Installation

  1. Download the Mac OS X disk image with the button below.
  2. Double-click the disk image to mount it.
  3. Drag KeyStore Explorer to your Applications folder.
  4. Eject the disk image
Download for Mac OS

Manual Installation

  1. Download the manual install archive with the button below.
  2. Extract the contents of the archive file somewhere and change into the kse directory.
  3. You can then use one of the following three methods to start KSE:
    • Execute kse.exe (on Windows)
    • Execute the shell script kse.sh (on Unix-like systems)
    • Use the command java -jar kse.jar
