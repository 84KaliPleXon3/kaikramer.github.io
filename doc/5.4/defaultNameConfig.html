Default Name Configuration

This allows to store default values for certificate distinguished names (DNs). When you create a new certificate, the input fields for Common Name, Organizational Unit, Organization etc. are pre-filled with the values from this configuration setting.

To configure the Default Name:

  1. From the Tools menu, choose Preferences.

    Mac OS users: From the KeyStore Explorer menu, choose Preferences.

  2. The Preferences dialog will be displayed. Select the Default Name tab.

  3. Enter any part of the DN that you normally use in your certificate DNs.

  4. Press the OK button to save your settings and dismiss the Preferences dialog.
